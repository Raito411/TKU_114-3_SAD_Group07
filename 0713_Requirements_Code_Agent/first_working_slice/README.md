# 第一個可操作切片

專案：校內二手書面交預約系統

## 操作方式

直接用瀏覽器開啟 `index.html`。

## 本次支援需求

| 需求編號 | 對應內容 |
| -------- | -------- |
| FR-01 | 買方可依書名、類別或關鍵字搜尋二手書。 |
| FR-03 | 買方可針對指定書籍送出購買需求。 |
| FR-04 | 賣方可新增一本二手書資料。 |
| NFR-01 | 書籍資料包含書名、類別、價格、書況、照片說明與交易狀態。 |

## 驗收方式

1. 在「賣方新增書籍」表單輸入書名、類別、價格、書況、照片說明與交易狀態。
2. 送出後，確認新增書籍出現在右側列表。
3. 在搜尋欄輸入書名、類別或關鍵字，確認列表會篩選。
4. 點選「查看詳細資料」，確認下方顯示書籍完整資訊。
5. 點選「送出購買需求」，確認畫面顯示「已送出，待賣方回覆」。

## 範圍外項目

本切片不包含正式登入、正式資料庫、線上付款、物流、跨校交易、手機原生 App 或真實身分驗證。
