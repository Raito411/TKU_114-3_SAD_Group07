# 需求文件包初稿

## 文件目的

```text
整理 0708 Project Charter v2 與 0713 需求來源、候選需求、第一個可操作切片成果，形成 0714 使用者故事、需求、驗收條件、追溯與實作修正的初稿。
```

## 範圍

```text
本期範圍內：
使用者註冊登入、書籍刊登、書籍搜尋、查看書籍詳細資料、提出購買需求、面交預約、交易狀態追蹤、資料庫保存、基本管理。

本期範圍外：
線上付款、第三方金流、手機原生 App、學校正式單一登入、正式學校帳號、真實身分驗證、物流寄送、跨校交易、自動估價、付費服務、正式部署。
```

## 角色與利害關係人

| 角色 | 說明 | 來源 |
| --- | --- | --- |
| 買方學生 | 搜尋二手書、查看書籍詳細資料、提出購買需求、確認面交時間與地點 | Project Charter v2 |
| 賣方學生 | 刊登二手書資料、管理書籍狀態、回覆購買需求、確認面交安排 | Project Charter v2 |
| 系統管理員 | 管理交易資料、處理不符合規範的刊登內容或買賣方資格、下架已完成交易 | Project Charter v2 |
