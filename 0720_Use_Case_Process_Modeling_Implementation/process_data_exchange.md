# 流程資料交換表

| 流程步驟 | 輸入資料 | 處理/規則 | 輸出資料 | 資料流向 |
|---|---|---|---|---|
| T-01 註冊或登入 | 使用者註冊或登入資料 | 建立或驗證帳號 | 使用者身份與登入狀態 | 使用者 -> 系統 -> 使用者 |
| T-02 新增書籍 | 書名、類別、價格、書況、照片說明、面交地點 | 狀態預設為未交易；產生不重複刊登序號 | 新書籍資料 | 賣方 -> 系統 -> 書籍列表 |
| T-03 設定折扣/編輯資料 | 折扣百分比、更新後書籍資料 | 以百分比顯示折扣，並顯示折扣後價格 | 更新後書籍資料 | 賣方 -> 系統 -> 書籍列表/詳細資料 |
| T-04 搜尋書籍 | 書名、類別或關鍵字 | 比對書籍資料 | 搜尋結果清單 | 買方 -> 系統 -> 買方 |
| T-05 查看詳細資料 | 書籍識別資料 | 檢查查看者是否為賣方本人或一般買方 | 書籍詳細資料；賣方自己的刊登不顯示購買按鈕 | 買方/賣方 -> 系統 -> 使用者 |
| T-06 送出或取消購買需求 | 書籍識別資料、買方身份、購買需求狀態 | 送出需求時通知賣方；未核准前可取消；取消需求時不通知賣方；取消後賣方查看要求同步移除該筆需求；公開狀態維持未交易 | 購買需求紀錄、賣方通知與提示狀態 | 買方 -> 系統 -> 買方/賣方信箱/賣方管理視圖 |
| T-07 選擇買方與面交時間 | 書籍序號、被選買方、面交時間 | 一本書一次只能核准一位買方；其他需求自動取消 | 書籍狀態交易中、通知紀錄 | 賣方 -> 系統 -> 買方信箱/交易狀態 |
| T-08 查看交易狀態/信箱 | 使用者身份 | 依身份顯示相關交易與通知；非交易當事人不顯示面交資訊 | 交易狀態、通知清單 | 使用者 -> 系統 -> 使用者 |
| T-09 完成或交易失敗 | 書籍序號、交易結果 | 完成由賣方更新；失敗則取消需求、重新刊登並扣信任度 | 交易完成或未交易狀態、信任度變更、通知 | 賣方 -> 系統 -> 買方/管理資料 |
| T-10 停權處理 | 信任度為 0 的使用者資料 | 系統管理員停權，取消其刊登書籍與購買需求 | 停權狀態、取消後資料 | 系統/管理員 -> 資料庫 |

## 資料項目來源

| 資料項目 | 來源文件 |
|---|---|
| 書名、類別、價格、書況、照片說明 | 0713、0714 需求與驗收條件 |
| 面交地點、面交時間 | 0714 後續修正需求 |
| 折扣百分比、折扣後價格 | 0714 後續修正需求 |
| 刊登序號 | 0714 後續修正需求 |
| 購買需求、要求人姓名、發送時間 | 0714 後續修正需求 |
| 交易狀態與信箱通知 | 0714 後續修正需求 |
| 信任度、停權資料 | 0714 後續修正需求 |
